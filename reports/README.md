# `reports/`

Long-form artefacts: technical reports (`TR-*.pdf`) and the figures
regenerated by the pipeline.

| Path | Contents |
| ---- | -------- |
| `TR-01.pdf` | Technical report 01 — project framing |
| `TR-02.pdf` | Technical report 02 — quantitative methodology |
| `TR-03.pdf` | Technical report 03 — Phase 1 results |
| `TR-04.pdf` | Technical report 04 — bridging Phase 1 → Phase 2, merge engineering, flag rationale |
| `TR-05.pdf` | Technical report 05 — qualitative database acquisition + flag-by-flag overview |
| `figures/` | PNG and HTML outputs of the training, analysis and inference scripts (incl. `inference_storyboard_2015.html`) |

The Markdown sources of TR-04 and TR-05 (`TR-04.md`, `TR-05.md`) are
also kept here so they can be edited without re-rendering the PDFs by
hand. Any text editor and `pandoc` (or the included reportlab
recipe in `scripts/`) can regenerate the PDFs.
